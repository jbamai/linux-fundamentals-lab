# linux-fundamentals-lab
A collection of Linux commands, exercises, and cybersecurity fundamentals.
## Commands Learned

### Navigation Commands

#### pwd

Displays the current working directory.

#### ls

Lists files and directories in the current location.

#### ls -a

Lists all files, including hidden files.

#### ls -l

Lists files and directories in a detailed format, showing permissions, owner, size, and modification date.

#### ls -la

Combines `-l` and `-a` to display all files, including hidden files, in detailed format.

#### cd

Changes the current working directory.

---

### File and Directory Management

#### mkdir

Creates a new directory.

#### rmdir

Removes an empty directory.

#### touch

Creates a new file.

#### cat

Displays the contents of a file.

---

### Linux Permissions

Linux permissions control who can access or modify files and directories.

Permission Types:

* `r` = Read
* `w` = Write
* `x` = Execute

Permissions are assigned to:

* User (Owner)
* Group
* Others

Understanding permissions is important for system security because it helps control access to files and programs.
